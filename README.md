## 📱 Responsive Design

This version is fully responsive across all devices:

| Breakpoint | Layout |
|---|---|
| Desktop (> 900px) | Full horizontal nav, multi-column grids |
| Tablet (≤ 900px) | Hamburger menu, 2-column grids |
| Mobile (≤ 600px) | Single-column, stacked buttons, compact spacing |
| Small Mobile (≤ 380px) | Further scaled typography and layouts |

**Mobile navigation:** A hamburger (☰) menu replaces the desktop nav bar on tablets and phones. Tapping it opens a full-width drawer with all page links.

## 🚀 Activating GitHub Pages

1. Push all files to your repository's **main** branch
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch, **/ (root)** folder → click **Save**
5. After 1–2 minutes your site will be live
